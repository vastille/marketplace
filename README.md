# Vastille Claude Code marketplace

> [!NOTE]
> The Vastille marketplace is in beta. Version 1.0 will be the first release we
> consider stable for general use.

This repository is a [Claude Code](https://docs.anthropic.com/en/docs/claude-code)
plugin marketplace. Plugins are declared in
[`.claude-plugin/marketplace.json`](.claude-plugin/marketplace.json).

## Plugins

- **vastille-source:** Agentic lower-middle market sourcing.
